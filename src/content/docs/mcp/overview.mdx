`devtoolkit-mcp` exposes all of DevToolKit's logic as a **Model Context Protocol (MCP) server**. Once connected, your AI assistant can call any of the 26 tools directly — no copy-pasting between windows.

## Quick Install

```bash
# Claude Desktop / Claude Code
npx devtoolkit-mcp
```

See [Setup & Installation](mcp/setup) for full configuration instructions per client.

## Why Use MCP Tools?

When you ask an AI to hash a string, generate a UUID, or calculate a subnet, it guesses. The MCP tools give **deterministic, correct results** using the same implementations as the browser app.
