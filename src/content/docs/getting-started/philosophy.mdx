## Local-First, Always

Every computation happens in your browser. No data is ever sent to a server — not even anonymised telemetry. Tools that need heavy processing (EXIF metadata extraction, PDF manipulation) use **WebAssembly** to run native-speed code locally.

## Privacy by Default

- No Google Analytics, Mixpanel, or any tracking script
- No cookies (only `localStorage` for your theme preference and optional API keys)
- No account required — ever
- The optional Gemini AI integration (Query Plan Viewer) uses your own API key, stored only in your browser

## One Input, Right Output

The **Smart Detector** is the default landing page. Paste anything and it routes you to the correct tool automatically. You should never have to remember which tool handles which format.

## Keyboard-Friendly

Every tool is designed to work without a mouse. Tab navigation, Enter to submit, Escape to cancel, `Ctrl+Enter` to trigger actions.

## Open Source & Auditable

The full source is on GitHub. If you want to verify that no data leaves your machine, read the code — there are no network calls except the optional Gemini API you opt in to.

<Aside type="note">
  The MCP server (`devtoolkit-mcp`) is a separate Node.js package that runs locally on your machine. It does not phone home.
</Aside>
