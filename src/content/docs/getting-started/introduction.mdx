DevToolKit is a collection of **25+ developer tools** that run entirely in your browser — no server, no account, no tracking.

## What It Solves

Developer workflows involve constant context-switching: formatting a JSON blob, decoding a JWT, building a cron expression, converting a timestamp. Most online tools for these tasks upload your data to a server, wrap it in ads, and track your usage.

DevToolKit does none of that. Every tool runs locally using browser APIs and WebAssembly.

## How to Navigate

Open the app and paste or type anything into **Smart Detector** — it auto-routes your content to the right tool. Or use the sidebar to go directly to any tool by name.
