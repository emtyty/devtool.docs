Binary Metadata extracts the full metadata payload from any file that contains structured metadata — images, PDFs, audio, video — using a WebAssembly build of [ExifTool](https://exiftool.org/), the industry-standard metadata library.

**Your files never leave your device.** ExifTool runs entirely in your browser via WASM.

## Supported File Types

| Type | Formats |
|------|---------|
| **Images** | JPEG, PNG, WebP, TIFF, HEIC, RAW (CR2, NEF, ARW, DNG…), GIF, BMP |
| **Documents** | PDF |
| **Audio** | MP3, FLAC, AAC, WAV, OGG |
| **Video** | MP4, MOV, MKV, AVI |
| **Vector** | SVG |

## Metadata Standards

| Standard | Description |
|----------|-------------|
| **EXIF** | Camera settings, GPS coordinates, timestamps |
| **XMP** | Adobe-standard extensible metadata (keywords, ratings, copyright) |
| **IPTC** | Press/editorial metadata (caption, credit, source) |
| **ID3** | Audio tags (artist, album, track, year) |

## How to Use It

1. **Drop a file** onto the drop zone (or click to browse).
2. All metadata fields appear in a searchable, grouped table.
3. **Filter by group** (EXIF, XMP, IPTC, etc.) using the tabs.
4. **Search** by field name or value.
5. **Copy** individual values, or **Export JSON** for the full payload.

<Aside type="caution">
**Privacy reminder:** JPEG photos from smartphones often contain precise GPS coordinates. Before sharing photos online, consider stripping EXIF data. Binary Metadata lets you inspect what's embedded — a future tool may add EXIF stripping.
